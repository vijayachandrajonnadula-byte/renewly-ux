# Visual Direction

## Visual Goal

Renewly should feel like a trustworthy B2B SaaS operations tool: calm, structured, readable, and efficient. The interface should support repeated work, comparison, and decision-making rather than feeling like a marketing page.

## Style Attributes

- Clean.
- Professional.
- Operational.
- Data-informed.
- Calm but not bland.
- High contrast for important status states.

## Layout Direction

- Use familiar dashboard and table patterns.
- Keep information dense but organized.
- Use cards and sections only where they group related content clearly.
- Avoid oversized hero sections in the product experience.
- Prioritize scanning, sorting, and action.

## Color Direction

The palette should support a neutral SaaS base with purposeful accents:

- Neutral backgrounds for work surfaces.
- Strong status colors for risk, approval, and opportunity.
- Clear contrast between normal renewals and high-priority items.
- Avoid relying on color alone to communicate state.

## Typography Direction

- Use a readable sans-serif.
- Keep headings proportional to dashboard and tool surfaces.
- Avoid overly large display text inside operational UI.
- Use consistent labels for statuses, actions, and categories.

## Interaction Direction

- Primary actions should be easy to find.
- Filters and sorting should follow familiar SaaS conventions.
- Details should reveal progressively.
- Confirmation states should feel explicit and reassuring.

## Accessibility Considerations

- Maintain strong text contrast.
- Use labels and icons together for important statuses.
- Avoid color-only meaning.
- Keep touch targets and clickable areas comfortable.
- Use clear focus and hover states in the future UI.
